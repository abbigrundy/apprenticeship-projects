Semantic HTML introduces meaning to the web page, it is best practice to use semantic HTML for improving communication, both with other developers you are collaborating with, and also for accessibility, as correct usage of semantic HTML will aid people who use screen readers in being able to navigate through a web page.

This is an example of a paragraph element, used for representing a paragraph. This can contain multiple different semantic elements within it. e.g. the strong tag, used to add emphasis.

Text elements can also be made: bold, italic, strikethrough, subscript, superscript
